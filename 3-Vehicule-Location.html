🪑 CONFIGURATION OPTIMALE DES SIÈGES

Conducteur : Ali (40 ans) | Passager avant : Ahmed (75 ans - espace jambes max)

⚠️ IMPÉRATIFS SANTÉ (Ahmed 75 ans) :
• Siège passager avant : obligatoire pour espace jambes et montée facile
• Appui-tête réglable : éviter torticolis
• Climatisation bi-zone : température différente avant/AR
• Suspensions souples : protéger colonne vertébrale
• Marchepied : faciliter montée/descente (hanches)

🎯 RECOMMANDATION POUR 4 PERSONNES

Location SUV 4x4 5 places type Toyota Land Cruiser Prado ou Mitsubishi Pajero Sport

  • Places : 2 avant + 2 AR (confort optimal, pas de 3e rangée inutile)
  • Coffre : 700-800 L (4 valises + eau + équipement)
  • 4x4 : Blocage différentiel (sable Tamanrasset–Djanet)
  • Climatisation : Bi-zone (avant pour père, AR pour mère/beau-père)
  • Sièges : Passager avant réglable en hauteur
  • Marchepied : Fixe ou rétractable (essentiel pour père)

3. PLANIFICATION CONDUCTEURS (2 conducteurs)

👥 Répartition

  • Ali (40 ans) : conducteur principal, 60% du temps
  • Jean (65 ans, médecin) : conducteur 2, 40% du temps, surveille santé
  • Ahmed (75 ans) : passager avant uniquement
  • Dahbia (69 ans) : passager AR gauche
Règles de sécurité :
• Durée max 3h consécutives (pause 30 min)
• Distance max/jour : 700 km
• Désert (Tamanrasset–Djanet) : alternance toutes les 2h
• Interdiction de conduite de nuit sur pistes
• Vitesse max 80 km/h (pistes)

4. ÉQUIPEMENT VÉHICULE & DÉSERT

✅ FOURNI PAR LOUEUR (à vérifier)

  • Pneus tout-terrain neufs + roue secours identique
  • 4x4 testé (blocage différentiel)
  • Climatisation bi-zone (vérifier froid AR)
  • Cric + manivelle + clés
  • Sangles remorquage (2x)
  • Palettes désensablage (2x)
  • Pompe air 12V
  • Jerrican essence 20L
  • Extincteur 2kg
  • Triangle + gilets (4x)

🎒 À APPORTER / ACHETER

  • GPS offline (Tracks4Africa)
  • Talkie-walkie (2x) pour désert
  • Téléphone satellite (location)
  • Glacière électrique 12V (médicaments père)
  • Bâche pare-soleil
  • Coussin lombaire (père)
⚠️ KIT SURVIE DÉSERT (obligatoire Tamanrasset–Djanet) :
  • Eau : 32L (8L/pers) + 1,5L/pers en bouteilles
  • Nourriture : rations 3 jours
  • Abri : tente ou bâche + couvertures survie (4x)
  • Médical : trousse complète (Jean vérifie)
  • Communication : téléphone satellite chargé

6. CONFORT SPÉCIFIQUE AHMED (75 ans)

  • Siège : passager avant, dossier relevé 15°
  • Appui-tête : réglé haut (nuque soutenue)
  • Ceinture : harnais ventral (pas sous-cutané)

  • Pauses : toutes les 1h30 (étirements)
  • Hydratation : gourde accessible (1L/2h)
